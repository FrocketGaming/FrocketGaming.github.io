Classification Regression

A model that finds the optimal hyperplane to separate classes.

Pros:

  • Effective in high-dimensional spaces
  • Versatile through different kernel functions

K-Nearest Neighbors (KNN)

Classification Regression

A model that classifies based on the majority class of nearest neighbors.

Pros:

  • Simple and easy to implement
  • No assumptions about data distribution
  • Can be used for classification and regression

Cons:

  • Computationally expensive for large datasets
  • Sensitive to irrelevant features
  • Requires feature scaling

Naive Bayes

Classification

A probabilistic classifier based on Bayes' theorem.

Pros:

  • Simple and fast
  • Works well with high-dimensional data
  • Requires little training data

Cons:

  • Assumes feature independence
  • Sensitive to feature selection
  • May be outperformed by more sophisticated models

Classification Regression

Cons:

  • Can be prone to overfitting

Clustering Algorithms

Unsupervised

Pros:

  • Discover hidden patterns

Cons:
